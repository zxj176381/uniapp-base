## Layout

每个页面滚动条，适配 webview

- 内嵌 `NavigationBar` 组件
- 内嵌 `PageStatus` 组件

### 使用方法

```js
<Layout>
  <view>container</view>
</Layout>;
//先引入组件
import Layout from "@/components/Layout/index.vue";
```

**注：可传入所有 `NavigationBar` 的 `props` 值【详情请查看 `NavigationBar` 的 `README.md`】**
**注：可传入所有 `PageStatus` 的 `props` 值【详情请查看 `PageStatus` 的 `README.md`】**
